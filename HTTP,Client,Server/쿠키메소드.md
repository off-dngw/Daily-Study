쿠키의 메소드 

메소드 | 설명
|---|--|
setCookie | 쿠키 저장하기
getCookie | 쿠키 가져오기
removeCookie | 쿠키 제거하기
name | 쿠키 이름을 구한다
value | 쿠키 값을 구한다 
Domain | 쿠키의 도메인을 지정한다.
Path | 쿠키의 전송 경로를 구한다.
MaxAge | 쿠키의 유효 시간을 구한다.

### 사용 방법
1. npm intsall react-cookie 패키지 다운로드
2. import { CookiesProvider } from 'react-cookie';

```
const [cookie, setCookie, removeCookie] = useCookies([]);

```
